AI-Powered Virtual Mouse and Voice Assistant

An AI-powered touchless human-computer interaction system that allows users to control their computer using hand gestures and voice commands. The project combines Computer Vision, Speech Recognition, and Desktop Automation to create a hands-free interface.

Gesture Mapping

Index Finger Movement -	Cursor Movement

Thumb + Index Finger Pinch	- Left Click

Thumb + Middle Finger Pinch	- Right Click

Pinch and Hold	- Drag & Drop

Voice Commands

Scroll Up	- Scroll Up

Scroll Down	- Scroll Down

Volume Up	- Increase Volume

Volume Down	- Decrease Volume

Left Click	- Left Mouse Click

Right Click	- Right Mouse Click
